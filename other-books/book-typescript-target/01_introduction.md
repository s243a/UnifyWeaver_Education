# Chapter 1: Introduction

The TypeScript target generates type-safe JavaScript from Prolog predicates.

## Installation

```bash
# Install TypeScript
npm install -g typescript

# Verify
tsc --version
```

## Loading the Target

```prolog
?- use_module('src/unifyweaver/targets/typescript_target').
[TypeScript Target] Initialized
```

## Target Information

Query the target's capabilities:

```prolog
?- target_info(Info).
Info = info{
  name: "TypeScript",
  family: javascript,
  features: [types, generics, async, modules],
  recursion_patterns: [tail_recursion, linear_recursion, list_fold]
}.
```

## First Example: Tail Recursion

### Prolog Source

```prolog
sum(0, Acc, Acc).
sum(N, Acc, Result) :- N > 0, N1 is N - 1, Acc1 is Acc + N, sum(N1, Acc1, Result).
```

### Compile Command

```prolog
?- compile_recursion(sum/3, [pattern(tail_recursion)], Code),
   write_typescript_module(Code, 'Sum.ts').
```

### Generated TypeScript

```typescript
// Generated by UnifyWeaver TypeScript Target
// Pattern: tail_recursion

export const sum = (n: number, acc: number = 0): number => {
  if (n <= 0) return acc;
  return sum(n - 1, acc + n);
};

// Strict version for guaranteed TCO
export const sumStrict = (n: number, acc: number = 0): number => {
  let current = n;
  let result = acc;
  while (current > 0) {
    result += current;
    current--;
  }
  return result;
};
```

### Running

```bash
npx tsc Sum.ts
node -e "const { sum } = require('./Sum'); console.log(sum(100));"
# 5050
```

## JavaScript Family

TypeScript is part of the JavaScript family:

| Target | Description |
|--------|-------------|
| `typescript` | Type-safe JS |
| `node` | Server-side JS |
| `deno` | Secure TS runtime |
| `bun` | Fast JS runtime |
| `browser` | DOM-based JS |
